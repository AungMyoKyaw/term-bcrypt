## Install

```shell
npm install -g term-bcrypt
```

## Usage

```shell
Usage
  $ bcrypt

Options
  --text
  --salt-rounds
  --help

Examples
  $ bcrypt
  $ bcrypt --text='myplaintext'
  $ bcrypt --salt-rounds='10'
  $ bcrypt --help
```
